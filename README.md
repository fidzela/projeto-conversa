# projeto-conversa

Chat "real-time" utilizando **Crocoblock JetEngine + JetFormBuilder + Elementor** no WordPress.

## 🔌 Plugin (versão atual)

A versão atual do projeto é o plugin [`conversa-chat/`](conversa-chat/README.md) —
substitui os 6 snippets do WPCode, integrado à raiz dos plugins em todas as
etapas. Arquitetura documentada em
[docs/07-plugin-conversa-chat.md](docs/07-plugin-conversa-chat.md).

Os arquivos `conversa-v*.php` na raiz do repositório são a **versão legada**
(WPCode), mantidos como histórico.
